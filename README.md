# Congressional-Chat-Room
Chat room website

### Description
A chat room where you can chat with other people or a bot

## So far:
- [X] Set up webserver
- [X] Set up basic flask script
- [ ] Set up to run with apache2

## How to Run
In terminal run:
```
python main.py
```

## Dependencies:
* Python 3

* Flask

## Note:
Make sure you are running python 3 with:
```
python --version
```
